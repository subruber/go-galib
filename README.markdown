Description
--------
Genetic Algorithms for Go/Golang

Install
--------
	$ go install git://github.com/thoj/go-galib.git

Compiling examples:

    $ git clone git://github.com/thoj/go-galib.git
    $ cd go-galib
	$ cd examples
	$ go build floating.go
    $ ./floating

Use
--------
See example/*
